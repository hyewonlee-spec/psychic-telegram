# Memory Calendar icon patch

Replace/add these files in your project:

- Replace `index.html`
- Add the `public/` folder and everything inside it:
  - `public/favicon.png`
  - `public/apple-touch-icon.png`
  - `public/icon-192.png`
  - `public/icon-512.png`
  - `public/site.webmanifest`

Then push to GitHub and redeploy on Vercel.
